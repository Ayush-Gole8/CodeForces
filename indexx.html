NeuroScan XAI: Transparent Brain MRI Tumor Analysis Through Explainable and Ethical AI

Abstract / Introduction

Brain MRI analysis is an important task in healthcare, but AI systems are often treated like final answers without showing how they reached a decision. This project addresses that problem by building a brain MRI tumor classification system that not only predicts the result, but also shows evidence, confidence context, bias signals, and an ethics report. The system classifies scans into glioma, meningioma, pituitary and no tumor.

Objectives

  • Classify brain MRI images accurately into specific tumor types.
  • Provide explainable predictions using generated heatmaps.
  • Reduce blind trust in model output by showing reasoning.
  • Make ethical risk visible through transparency and fairness reporting.
  • Serve as an educational decision-support prototype rather than a clinical diagnostic device.

Methodology

EfficientNet-B0 is used as the base classifier because it is a parameter-efficient model with a good accuracy-to-compute balance. Grad-CAM++ generates explainable heatmaps for visual verification.

Workflow:
  • MRI Upload
  • Preprocessing
  • EfficientNet-B0 Model Inference
  • Prediction Output & Class Probabilities
  • Grad-CAM++ Heatmap Overlay
  • Confidence & Entropy Context
  • Bias / Transparency Analysis
  • Ethics Dashboard / Final Report

Results

The API provides MRI prediction with probabilities, Grad-CAM++ overlays, and a bias audit. The ethics report includes transparency metrics, a fairness summary, a saliency audit, a model card, and mitigations.

Conclusion

  • Demonstrates that medical AI should provide reasons alongside predictions.
  • Improves trust, interpretability, and accountability in brain MRI analysis.
  • Successfully combines classification, explainability, and ethics reporting.
  • Functions as a responsible AI prototype designed for educational use.

Future Scope

  • Train on larger and multi-source datasets to expand data diversity.
  • Improve model generalization across different MRI scanners and institutions.
  • Add classification capabilities for more tumor classes.
  • Integrate expert review for stronger real-world validation.
